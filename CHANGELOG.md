# Changelog

本文件记录 `report-helper` 的公开版本更新。

## v2026.06.02 - 2026-06-02

### 新增

- 增加 README 安装说明，提供可直接复制给 Codex agent 的安装请求。
- 增加 `CHANGELOG.md`，用于记录公开版本更新。
- 增加公司研究的成长性与估值定量分析要求，覆盖收入/ARR、用户或客户、利润率、现金流、融资、估值和可比公司指标。
- 增加写作前「最新数据校验」要求，进入正文前必须核查近 12-24 个月的官方披露、监管文件、财报、原始研究或权威报道。

### 改进

- README 示例区补充运行截图、PDF 预览图和示例报告。
- 示例 PDF 改为用户提供的原始版本，不再追加修改。
- 补充 MIT License。
- 强化 `gotchas.md` 维护规则：用户指出的可复用流程、事实或结构问题必须及时入库。

## v2026.06.01 - 2026-06-01

### 新增

- 发布 `report-helper` 公开版。
- 支持深度研究报告工作流：研究范围确认、资料搜集、信息充分性审计、写作准备审计、正文写作、审核和 PDF 生成。
- 增加来源分级和正文上标引用规则，要求关键事实判断与文末来源编号一一对应。
- 增加 PDF 排版脚本和 Chrome fallback 渲染路径。
- 增加本地配置模板 `config.example.json`，支持输出目录、中间资料目录和报告署名配置。

### 调整

- 公开版聚焦 PDF 输出，EPUB 输出移除。
- 内部 Markdown 仅作为 PDF 渲染构建稿，不作为公开交付物。
